1. Изменение количества бонусов можно производить как с помощью мероприятий
   (для нескольких учеников), так и через таблицу класса (для отдельного ученика).
2. При редактировании одной ячейки таблицы, нажимайте после её изменения 
   кнопку "Enter" для сохранения данных.
3. Остальные данные сохраняются автоматически, дополнительно нажимать
   ничего не нужно.
4. Для добавления мероприятия не обязательно вводить описание и участников, 
   это можно будет сделать позже через меню редактирования мероприятия.
5. При вводе участников мероприятия соблюдайте следующий формат:
   'Фамилия имя отчество класс', например: 'Иванов Иван Иванович 11-1' (без кавычек). 
   Фамилия, имя, отчество и класс разделяются одним пробелом.
   Не должно быть лмшних пробелов в строке и на её конце.
6. После проведения мероприятия с помощью кнопки 'Отметить проведение' можно удалить
   это мероприятие и начислить всем его участникам некоторое количество бонусов.
   Это значение должно представлять собой целое неотрицательное число.
